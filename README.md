# Avaliação Automatizada de Aptidão Física com a <br/> Câmera Orbbec Femto Mega

Seguindo a bateria de testes de aptidão física de Fullerton, avaliar os exercícios por meio da <a href="https://doc.orbbec.com/documentation/Orbbec%20Femto%20Mega%20Documentation"> *Câmera Orbbec Femto Mega* </a>.

Levando em consideração o <a href="https://github.com/vgguerra/Propicie"> trabalho dessenvolvido por Victor Guerra </a> e posteriormente <a href="https://github.com/JuliaKoene/Propicie_visual"> editado por Julia Koene e Diogo Castro </a>, foram repensados a lógica e funcionamento para utilizar a Câmera Orbbec em busca de melhores resultados.
